Devenez Développeur Web, et forger votre avenir


Voici ce que tkt.paris raconte concernant les avantages du développement informatique : Toutes les personnes qui apprennent la programmation n’auront pas un jour de créer le nouveau Facebook. Toutefois, savoir coder vous donne la possibilité de faire naître un projet par vous-même. Si vous avez l’âme créative et que vous souhaitez innover en créant un site, une application ou bien encore un logiciel, vous aurez la capacité de le faire et d’apporter au monde votre contribution. Si vous souhaitez devenir patron de votre propre entreprise, connaître les bases fondamentales du développement web ne sera pas de trop.
Nous ne voullons pas seulement vous formez et vous abandonnez a votre propre sort mais notre but est que vous parvenez soit a trouver un boulot soit a entreprendre, voila pourquoi nous mettons a votre disposition un incubateur pour vous accompagnez et faire de vous un développeur Web agueri.



Le type de Diplôme que vous obtiendrez


Cette formation vous permet d'obtenir un diplôme de type Bac + 4 en seulement six (6) mois plus les deux (2) mois et une semaine pour les stages et la preparation pour la recherche d'emploi en ligne et local.



Pré-requis

La seule et unique conditions avant de commencer cette formation est d'avoir un Baccalaureat et un quelque connaissance en Informatique.



Le contenu de ce que vous allez apprendre figure sur la liste si-dessus



A/ Introduction

Durant cette partie longue et theorique nous allons vous inculquez toutes les notions necessaires pour que vous vous lancer dans le métier du Web sans lacune. Vous allez donc apprendre quelque notions de reseau que nous jugons necessaire tel que le modele OSI, le TCP/IP, le DNS, l'architecture client-serveur, HTTP/HTTPS, FTP, SSH, NTP, Telnet.


B/ Les languages et formats du Web

Ici vous allez apprendre la difference entre XML et JSON. En suite vous allez apprendre de facons pratique a utilisés tous les tags du HTML5 les plus utilisés.


C/ Styliser HTML5 avec CSS3

Durrant cette partie toujours pratique de la formation, vous allez apprendre a styliser les tags HTML5 que vous avez appris precedement, ensuite vous allez apprendre quelques notions sur bootstrap qui l'un des framework CSS les plus populaires. A la fin de cette partie vous aurez deja realisé un site web static.


D/ Dynaminer HTML5 avec Javascript

Durant cette partie nous allons continuer a travailler sur le site que aurez réaliser et le but est de dynamiser les éléménts HTML et de ce site grace a Javascript et particulierement a traver Jquery qui est un framework de Javascript.


E/ Choisir entre Python/Django, Java/Springboot, et PHP/Laravel

Selon le choix que vous allez faire, vous étudierez d'abord le language de programmation choisis ensuite le framework qui correspond. Vous allez apprendre a realisé des Rest API et recuperé les données a partir ces API et les afficher dans vos view (vue). Vous allez également apprendre a travaillé avec les bases de données tels que: Mysql, Mongodb.
