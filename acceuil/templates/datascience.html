L'intelligence Artificielle, au coeur de l'innovation


utilisé dans toutes les grandes entreprises Itech, l'IA notamment la machine learning est tout simplement le sujet le plus brulant en informatique. L'intelligence artificielle est utilisé dans quasimment tous les domaines. Tous le monde veut profiter des avantages qu'elle a la disposition de l'humanité. A travers cette formation pratique nous voulons vous apprendre a implémenter vos propres modeles a partir de zero.



Le type de Diplôme que vous obtiendrez


Cette formation vous permet d'obtenir un diplôme de type Bac + 4 en seulement six (6) mois plus les deux (2) mois et une semaine pour les stages et la preparation pour la recherche d'emploi en ligne et local.



Pré-requis

Il y a certaines conditions requises avant de commencer cette formation. La premiere condition c'est vous devriez avoir au minimum un Baccalaureat scientifique de préférence. Vous devriez aussi avoir une parfaite maitrise du language de programmation Python.



Le contenu de ce que vous allez apprendre figure sur la liste si-dessus



A/ Initiation a l'IA

Durant cette partie longue et theorique vous allez apprendre a prepondre aux questions suivantes:
  • Qu'est que l'intelligence artificielle ?
  • Quel sont les differents approches du IA ?
  • Quelle est a la difference entre l'IA, la machine learning, le deep learning et la data science ?
  • Quelles sont les domaines ou l'IA est appliqué et de quelle maniere ?
Vous allez également apprendre a classifié les differents types d'algorithmes du machine learning en fonctions des problemes que ses derniers regles. Ces connaissances vous permettrons d'acquerire une maitrise theorique de l'IA en général.


B/ Comment la machine apprend t-elle ?

Avant d'apprendre a implémenté un modele de machine learning, en suite l'entrainé et le tester, il est crucial de d'abord comprendre de quelle maniere la machine apprend. Ces connaices vous permettrons de mieux regler les problemes que vous allez rencontrer au cours du développement de votre modele. Vous allez aussi apprendre les differents types d'apprentissage et ce qui les differencies.


C/ Les maths pour l'IA

Durrant cette partie vous allez apprendre quelques notions de bases des mathematiques lié a l'IA. Tel que les probabilités en statisque, les calculs sur les matrixes en Algebre linéaire et enfin les fonctions. Nous n'iront pas en profondeurs le but ici est juste de vous faire comprendre de quelle maniere les maths sont liés a l'intelligence artificielle afin que vous ne soyez pas confus devant certaines termes mathematiques utilisés dans la plus part des algorithmes du machines learning.


D/ Les Instruments du data scientist

Dans plusieurs domaines mais tous, le role du data scientis est de se servir de L'IA pour faire de la prediction. Mais quelles instruments utilise t'il pour implementer ses modeles ? C'est a cette question que nous allons repondre ici. Vous allez d'abord apprendre des notions requises pour dévélopper un modele et ensuite les instruments qu'il faut tels que: conda, anaconda, Tensorflow, Scikit-learn, keras, etc...


E/ Les Algorithmes du machine learning

Cette partie tres pratique vous permettra d'apprendre le fonctionnement de toutes les algorithmes du machines et Comment les implementer.


F/ Le deep leaning et ses particularités

Le deep learning est une forme de machine learning mais ayant ses propres particularités Dans cettes parties vous allez apprendre ces particularités. Et vous allez apprendre les amgorithmes du deep learning tels que: reccurent neural network, convnet.


G/ Deep learning en pratique

A la fin de cette formation vous allez mettre le deep learning en application avec les fichiers audio et les images.
